## **Factions**
* Added Early War USMC
* Added Early War IJA

## **Weapons**
* Added Type 89 Grenade Discharger (Knee Mortar) to IJA Mortar, with proper voice line warnings
* Added M38 Carbine to RKKA Engi Early War
* Added M44 Carbine to RKKA Engi Mid War (Replaces M38)
* Added PPSH 41 early variant to early war RKKA loadouts
* Added M1 Carbine to mid-war US Rifleman loadouts
* Added G43 to FJ Rifleman
* Elite Rifleman FJ now only use FG42
* RKKA Engis no longer have SMGs
* Made bolting the same speed for all bolt action weapons (Overall 15-30% speed increase in bolting depending on firing mode)
* Weapon FOV is now fixed to the sight rather than depending on shift zoom
* Some weapons recieved Weapon FOV changes (Arisakas, G41, G43, etc.) to allow better front sight visibility
* Fixed various weapon dialing issues
* Fixed sensitivity issues for aiming
* Fixed an issue where the Type 97 Scoped did not have bayonet functionality
* Fixed an issue where you could not use the sights on an MG when bipoding on a surface
* Fixed M1 Garand not making a ping on last shot
* Fixed PTRD causing crashes
* Updated M3 Grease Gun sights
* Updated Type 99 sights

## **Gameplay**
* Added Nebelwerfer sounds for German Rockets
* Fixed Katyusha rockets not playing properly
* Banzai speed buff is now capped (15% boost max)
* Banzai damage resisitance is now fixed, and can stop damage up to 95%
* Banzai damage resistance now only applies to non lethal damage. They can tank critical hits now.

## **Maps**
* Added map Barashka (Eastern Front)
* Added map Guadalcanal (Pacific)
* Added map Grain Elevator (Eastern Front)
* Added map Horson (Eastern Front)
* Added map Port Brest (Western Front)
* Added new spawn protection where it kills the player like in RO2
* Added RS2 wind volumes to the maps so smoke grenades work properly
* Added recon to Bridges, Carentan and Port Brest
* Added US Radioman to maps where US attacks
* Fixed artillery cooldowns
* Fixed customization mannequins
* Fixed placeholder static radio meshes
* Fixed missing Japanese HMGs on Kwajalein and Peleliu
* Various other map fixes that were too small to list

### **Iwo Jima**
* D and E objectives now disable on capture
* Decreased timer by 5 minutes (30 minutes to 25 minutes)

### **Brecourt**
* Further optimization fixes
* Replaced tents with better collision

* Additional trenches

## **Bots**
* Added experimental vaulting
* Added experimental path finding (less teleportation)
* Added more realistic firing intervals (No more full auto M1 Carbine)
* Bots prioritize body shots more than headshots at lower distances

## **Other**
